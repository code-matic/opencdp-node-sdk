# Changelog

All notable changes to the OpenCDP Flutter SDK.

## Version History

For the complete version history and changelog, please visit:

## Current Version

Latest stable release includes:

### Features
- User identification and attribute tracking
- Custom event tracking
- Screen view tracking (manual and automatic)
- Push notification delivery and open tracking
- Device token registration (FCM/APNS)
- Application lifecycle event tracking
- Customer.io dual-write support
- Debug mode for development

### Platform Support
- iOS 11.0+
- Android (API 21+)
- Firebase Cloud Messaging integration
